#XUE.JS

> v1.0.0

[XUE.JS](http://xesui.com/) 是学而思网校的前端框架。其中包括网校的样式解决方案，UI组件的交互效果，技术文档，以及一整套学而思网校前端架构解决方案。目前还处于测试阶段，仅供内部使用。

* 当前版本：v1.0
* 最后更新：2015-4-22
