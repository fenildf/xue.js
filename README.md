#XUE.JS

## 版本说明
> 当前版本：v1.0.0 （原XESUI v0.5 版）
>
> 更新日期：2015/4/22


##主题：蜕变：
1. 名称：2015年 xesui更名xue.js
2. Logo：由原来的字母加树叶转变为蝴蝶造型的X

## 介绍

1. 前身是XESUI，即原学而思网校前端框架；
2. 现改为XUE.JS 致力于学而思网校前端解决方案，其中包括三个方面：
	+ 自动化工具、性能优化、模块化框架、开发规范、代码部署、开发流程等；
	+ HTML5跨屏前端框架；
	+ 构建WEB应用
